# 널 병합 연산자

널 병합 연산자 ??는 연산자 좌측의 값을 먼저 검사해 null 혹은 undefined가 아닐 경우 좌측의 값을 반환하고, 그렇지 않다면 우측의 값을 반환한다.

```js
console.log(2 ?? 1) // 2
console.log(undefined ?? 1) // 1
console.log(null ?? 1) // 1
console.log(null ?? undefined) // undefined
```

아래와 같이 대입연산자와 함께 사용할 수 있다.

```js
let [a,b] = [null, null];

a = a ?? 1;

b ??= 2;

console.log(a, b) // 1 2
```
