# startsWith()

특정 문자열이 어떤 문자열로 시작하는지 여부를 확인하고 true, false를 반환한다.


```js

const str = "string";
const str2 = "str";

console.log(str.startsWith(str2)) // true

```

# endsWith()

특정 문자열이 어떤 문자열로 끝나는지 여부를 확인하고 true, false를 반환한다.

```js

const str = "string";
const str2 = "ing";

console.log(str.endsWith(str2)) // true

```
